Detailed Description

template<typename V>
class mvcc::SkipList< V >

A typically SkipList provides concurrent write and read.

Class SkipList is a partly thread safe container, whose underlying data structure is skip list. The value type should has default constructor or copyable.

Template Parameters
VValue type of skip list node
Note
The operations on different skip list nodes are thread safe, but the update operations on same node is not thread safe. User can use a MVCC strategy to make update of value thread safe.
