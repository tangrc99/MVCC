Detailed Description

template<typename V>
class mvcc::SkipList< V >::Iterator

Iterator of SkipList.

class Iterator is used to warp SkipListNode. The CRUD operation will return Iterator to hide underlying data structure details. User can also use iterator to traverse SkipList on the bottom level thread safely.
