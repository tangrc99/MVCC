Detailed Description

BulkWriteOperation derived from Operation. Generated by OpCoordinator.

Class BulkWriteOperation is an abstract of bulk write process. User should firstly use function appendOperation to append write operation to this impl and then use function run to start a bulk write. Every write operation will shares the same version.

Note
Bulk write process will be shutdown when facing an error without rollback.
